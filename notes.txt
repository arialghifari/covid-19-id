Karena api yang dipakai menggunakan Same-Origin Policy,
maka untuk mem-bypass saya menggunakan "moesif origin cors"
pada google chrome dan WAJIB untuk diinstall dan diaktifkan ketika menjalankan applikasi:
https://chrome.google.com/webstore/detail/moesif-origin-cors-change/digfbfaphojjndkpccljibejjbppifbc

Running for development:
npm start / npm run start

Running for production:
npm run build
